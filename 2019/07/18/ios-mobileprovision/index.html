如何编写iOS的通用构建脚本？

Posted on 2019-07-18 Edited on 2021-01-26
Symbols count in article: 2.1k Reading time ≈ 2 mins.

背景

最近在负责平安银行内部的MCD项目开发，目标是为银行体系内部提供持续化的集成发布平台，为移动端及H5端的提供构建、发布等相关功能。

目标

由于做的是平台化的项目，平安银行体系内已经存在的iOS项目接近20个左右，后续随着业务的开发，移动端的项目肯定还是会持续的递增，因此为了满足需求同时保证后续脚本的可维护性，因此尽量保证脚本的通用性。

实现方式

在iOS的持续话集成的开发过程中，证书的配置是不可避免的问题。因此在开发通用脚本的过程中，同样证书配置也成为了拦路虎。

按以往常规的思路来讲，证书的配置一般都会在工程中或者在脚本中配置相关的证书信息，但是平安银行内部的测试包分发都是走的企业证书签名，因此每个项目基本都会存在两套及以上的证书配置。

因此需要另辟蹊径，经过一番需求的分析及调研，最终的思路是通过用户在入住时上传不同的证书来实现不同的证书签名，不需要变更脚本。

通用脚本

如下为具体的python脚本解析mobileprovision文件，获取证书相关的信息。

#!/user/bin/python
# -*- coding: utf-8 -*-
import re, os, commands

class mobileprovision(object):

    @classmethod
    def get_mobileprovision_info(self, file_path):

        xml_string = commands.getoutput(' '.join(['security cms -D -i', file_path]))
        xml_list = xml_string.split('\n')
        uuid,name,bundle_id,team_id = [None] * 4
        for line in range(len(xml_list)):
            if uuid and bundle_id and name and team_id:
                break
            else:
                if None == uuid:
                    re_uuid = re.search(r'<key>\s*UUID\s*</key>',xml_list[line])
                    if re_uuid:
                        temp = xml_list[line + 1]
                        temp = temp.strip()
                        uuid = temp.replace('<string>', '').replace('</string>', "")
                if None == bundle_id:
                    if 'application-identifier' in xml_list[line]:
                        bundle_id = xml_list[line + 1]
                        bundle_id = re.search(r'com.[\w]*.[.\w]*',str(bundle_id)).group()
                if None == name:
                    re_name = re.search(r'<key>\s*Name\s*</key>',xml_list[line])
                    if re_name:
                        temp = xml_list[line + 1]
                        temp = temp.strip()
                        name = temp.replace('<string>', '').replace('</string>', "")
                if None == team_id:
                    if 'com.apple.developer.team-identifier' in xml_list[line]:
                        temp = xml_list[line + 1]
                        temp = temp.strip()
                        team_id = temp.replace('<string>', '').replace('</string>', "")

        return {bundle_id: dict(uuid=uuid,name=name,team_id=team_id)}

传入用户上传的company_Dev.mobileprovision路径
